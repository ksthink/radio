🔐 YouTube Music 인증

⚡ 최간단 방법 (추천):

  1. 로컬 PC 터미널에서 실행: python3 get_youtube_headers.py
  2. 나오는 JSON을 전체 복사
  3. 아래 텍스트박스에 붙여넣기
  4. "YouTube 로그인" 버튼 클릭

📖 수동 방법 (개발자 도구 사용):

  1. 로컬 PC에서 YouTube Music 웹사이트 방문: https://music.youtube.com
  2. Google 계정으로 로그인
  3. 브라우저 개발자 도구 열기: Windows/Linux: F12 또는 Mac: Cmd+Option+I
  4. 'Network' 탭에서 any YouTube request의 'Request Headers' 확인
  5. Authorization 또는 Cookie 헤더 복사
  6. 아래 JSON 형식으로 작성하여 붙여넣기

✓ 올바른 JSON 형식:

방법 A) SAPISID:
{
  "Authorization": "SAPISIDHASH [복사한값]",
  "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
}

방법 B) Cookie:
{
  "Cookie": "[복사한쿠키값]",
  "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
}
